# Agent-Era Architecture Portfolio — deploy guide

Live site: **https://rjslogar.github.io**

Three filed provisional patent briefs on enterprise architecture for the
AI-agent era, written in plain English:

1. **Published Is Not a Policy** — governed content delivery to agents and humans
2. **Two Cohorts, One Metric** — customer intelligence where the audit record
   and the causal training data are one structure
3. **Bias Doesn't Stay Where It Started** — post-inference comprehensibility
   scoring with propagation containment

Each brief ships as governed siblings: an HTML page for people and a canonical
markdown twin for agents, indexed in [`llms.txt`](llms.txt). The site
demonstrates the delivery pattern the first brief argues for.
